Capstone Project (Big Greasy)
Unity — C# | XR/VR
List of Contributions:
Level Design
Contributed to the level design of "Big Greasy" by:
  • Exploring player interactions with the world and creating engaging level layouts.
  • Implementing NPCs with text and voice outputs (including recording the voice lines) for the player to interact with.
  • Ensuring the level design supports and enhances the game's narrative and mechanics — including random objects the player can pick up and throw.
UI Developer
Developed the user interface for "Big Greasy," focusing on:
  • Creating interactions between the player and the world using an event-driven interface that avoids constant polling in the Update function.
  • Developing dynamic UI elements that provide information to the player about interactable objects.
  • Building the Rat Race UI — a highlight of this project. Coordinating between a UI Manager, the RaceTrack script (which determines race start and winner), and the RatMenu script (which manages player money) required careful system design and was the most rewarding work on this project.
