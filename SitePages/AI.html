Flocking Algorithm Implementation
Implemented flocking algorithms to simulate realistic group behaviors, including:
  • Alignment: Calculated the average velocity of each boid and scaled it to create a unit-length influence vector.
  • Cohesion: Created an influence vector from each boid's position to the average position of the flock, scaling it based on distance.
  • Separation: Calculated vectors away from nearby boids to avoid collisions, scaling the influence based on proximity.
  • Combined these behaviors to achieve lifelike flocking movement and tested the implementation for various scenarios.
