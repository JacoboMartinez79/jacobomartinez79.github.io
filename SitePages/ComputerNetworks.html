C++ | WinSock API
List of Contributions:
Server Implementation
Developed and implemented the server-side application using C++ and the WinSock API, which included:
  • Setting up the server to handle multiple client connections simultaneously — the current limit is 5.
  • Implementing data serialization and deserialization for efficient network communication. Data is saved to a text file.
  • Ensuring robust error handling and connection management.
Client Application
Created the client-side application with C++ and the WinSock API, focusing on:
  • Developing a user-friendly interface for clients to connect to the server.
  • Implementing features for sending and receiving messages in real-time.
  • Ensuring the client application handles unexpected server disconnections gracefully.
